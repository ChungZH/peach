# 🍑 Peach

> 你在想屁吃？

Peach is a sweet Scoop bucket.

## Usage

```powershell
scoop bucket add peach https://github.com/ChungZH/peach
# Recommended, but you can omit the bucket name most of the time
scoop install peach/<app>
```
